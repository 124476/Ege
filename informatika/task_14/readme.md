### Перевод из 10 сс в x сс
```
def f(n, x):
    a = []
    while n > 0:
        a += [n % x]
        n //= x

    return a[::-1]

print(f(173, 16))
```

### Последняя цифра в сс

```

x // n % n # Предпоследняя цифра
x // n ** 2 % n # Предпредпоследняя цифра
```
